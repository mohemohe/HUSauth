HUSauth
=======

### これ is 何

北海道科学大学の学内ネットワークに接続する際に必要なWeb認証を自動で行いたいです  
もしかしたらGoNETで認証を行っている他のネットワークでも使用できるかもしれません

今のところはアカウント情報の記憶と1クリックログインができます  
クソ重い大学のトップページに飛ばないだけでもストレスが減ると思います

### 必要な物

- .NET Framework 4.5
- （開発に参加する場合は）Livet
